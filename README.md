## 🚀 How to Run this Project Locally

Since this is a static website containing HTML, CSS, and client-side JavaScript, you do not need to install any heavy compiler frameworks or dependencies. You can run it in a few ways:

### Option 1: Live Server (Recommended for Developers)
If you are using **VS Code**, install the **Live Server** extension:
1. Open the folder in VS Code.
2. Click the **Go Live** button in the status bar at the bottom right.
3. Your default browser will open the site automatically at `http://127.0.0.1:5500/index.html`.

### Option 2: Using Node.js (`npx serve`)
If you have Node.js installed on your system:
```bash
# Run this command in the project root directory
npx serve
```
Then, open `http://localhost:3000` in your web browser.

### Option 3: Using Python
If you have Python installed:
```bash
# For Python 3.x
python -m http.server 8000
```
Then, open `http://localhost:8000` in your web browser.

### Option 4: Direct File Execution
You can also open the portfolio by double-clicking the `index.html` file in your file explorer. However, note that some features (like dynamic fetching or modular imports if added in the future) might encounter CORS policies in local files. Using Option 1, 2, or 3 is highly recommended.

## ✨ Features & Highlights

- **Custom Interactive Cursor**: Smooth-following custom cursor dot and ring that react to hover targets.
- **Dynamic Loading Screen**: Premium intro sequence tracking load percentage before revealing page content.
- **Dual-Focus Skills Matrix**: Highlights competency in both IT/Software (MERN stack, Python, Agentic AI) and Core/Hardware (PIC18F4550, IoT, Proteus, Embedded C).
- **Fully Responsive & Interactive Cards**: Modern layout using CSS Grid/Flexbox, equipped with custom vanilla JS tilt effects (`data-tilt`) and hover triggers.
- **Scroll-Triggered Animations**: Smooth entry animations for text and layouts as they enter the browser viewport.
- **Interactive Resume Dropdown**: Quick selection of specialized software vs. hardware resumes.
